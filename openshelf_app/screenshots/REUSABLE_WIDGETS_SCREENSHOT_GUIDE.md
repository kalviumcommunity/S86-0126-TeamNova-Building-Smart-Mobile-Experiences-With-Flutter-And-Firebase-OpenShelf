## ðŸŽ¥ Video Demo Guidelines

### Duration: 1-2 minutes

### Content to Cover:

1. **Introduction (10 seconds)**
   - "Hi, I'm demonstrating reusable custom widgets in our OpenShelf app"
   - Show the app home screen

2. **Navigate to Demo Screen (15 seconds)**
   - Navigate to `/reusable-widgets-demo`
   - Scroll through showing all widget sections

3. **Explain Each Widget Type (30 seconds)**
   - Briefly mention each of the 5 widget types
   - Show how they differ (colors, styles, configurations)

4. **Navigate to Library Screen (20 seconds)**
   - Navigate to `/library`
   - Scroll to show multiple uses of the same widgets
   - Point out: "Notice how the same StatsCard is used 4 times"

5. **Demonstrate Interaction (15 seconds)**
   - Tap a book card to show favorite toggle
   - Click a custom button to show action
   - Tap a feature tile

6. **Conclusion (10 seconds)**
   - "These 5 custom widgets are reused 30+ times across the app"
   - "This improves consistency and reduces code duplication"

## ðŸ“± How to Capture Screenshots

### On Android Emulator:

1. Run the app in Android emulator
2. Navigate to desired screen
3. Click the camera icon in emulator toolbar
4. Screenshots saved to: `~/Pictures/` or use the share feature

### On iOS Simulator:

1. Run the app in iOS simulator
2. Navigate to desired screen
3. Press `Cmd + S` to save screenshot
4. Screenshots saved to: Desktop

### On Physical Device:

**Android:**

- Press `Power + Volume Down` simultaneously
- Find in Gallery or Screenshots folder

**iOS:**

- Press `Side Button + Volume Up` simultaneously
- Find in Photos app

## ðŸŽ¬ Video Recording Tools

### Recommended Tools:

1. **Screen Recording (Built-in)**
   - **Windows:** Windows + G (Game Bar)
   - **Mac:** QuickTime Player > New Screen Recording
   - **Android:** Built-in screen recorder
   - **iOS:** Control Center > Screen Recording

2. **Loom** (Recommended)
   - Free tier available
   - Easy sharing
   - Automatic upload to cloud
   - Get link immediately

3. **OBS Studio** (Advanced)
   - Free and open-source
   - Professional quality
   - More control

### Video Format:

- **Format:** MP4 (most compatible)
- **Resolution:** 720p or 1080p
- **Upload to:** Google Drive, Loom, or YouTube (unlisted)
- **Share settings:** "Anyone with the link"

## ðŸ’¡ Tips for Great Screenshots

1. **Clean Display**
   - Remove any debug banners
   - Use a clean emulator/device
   - Ensure good lighting for physical devices

2. **Highlight Key Features**
   - Capture varied widget states
   - Show different configurations
   - Include interactive states

3. **Consistency**
   - Use same device/emulator for all screenshots
   - Maintain consistent screen orientation
   - Use same app theme

4. **Context**
   - Show full screen when possible
   - Include navigation elements
   - Demonstrate real-world usage
